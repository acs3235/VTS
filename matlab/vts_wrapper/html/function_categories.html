Monte Carlo

Solvers

  • ROfFxAndFt - Reflectance as a function of spatial- and temporal- frequencies
  • ROfFxAndT - Reflectance as a function of spatial-frequency and time
  • ROfRho - Reflectance as a function of rho
  • ROfRhoAndFt - Reflectance as a function of rho and temporal-frequency
  • ROfRhoAndT - Reflectance as a function of rho and time
  • SetSolverType - Sets the solver type
